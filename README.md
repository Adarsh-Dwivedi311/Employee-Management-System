[EMPLOYEE-MANAGEMENT-SYSTEM]

📌 Description

This is a Java-based Employee Management System developed using Swing and MySQL. It helps in managing employee records efficiently.

>Add Employee

>Update Employee

>Delete Employee

>Search Employee

>Login Authentication
